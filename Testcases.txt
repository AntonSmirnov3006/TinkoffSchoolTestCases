Предусловия ко всем тест-кейсам: 
1. зайти на страницу, расположенную по URL-адресу:https://www.tinkoff.ru/mobile-operator/
2. тапнуть на “Заказать сим-карту”

1. Взаимодействие с полями ввода

1.1. Ввод данных: оповещение о не правильности данных
Шаги воспроизведения( далее “ШВ”):
нажимаем на поле ФИО (во всех полях вписаны подсказки вводимой информации)
убрать фокус с поля
  Ожидаемый результат ( далее “ОР”):
По периметру поля появляется красный контур а также подсказка что делать
  Фактический результат ( далее “ФР”):
По периметру поля появляется красный контур а также подсказка что делать
Тот же результат в других полях кроме “Гражданства”

1.2. Ввод данных: оповещение о правильности данных
ШВ:
Нажимаем на поле ФИО и вводим корректные данные, пример (Смирнов Антон Юрьевич)
Нажимаем на другое поле или просто на страницу без перехода
ОР:
Появится подтверждение ввода корректных данных путем появления зелёной галочки 
ФР:
Появится подтверждение ввода корректных данных путем появления зелёной галочки 

1.3. Ввод данных: попытка оставить заявку в регионе где нет доставки
ШВ:
вводим все корректные данные в предложенные поля по образцу
нажимаем на кнопку “Оставить заявку”
ОР:
Переходит на форму где появится сообщение о результате ( в нашем случае говорится что “ нас пока ещё нет в вашем регионе”)
ФР:
Переходит на форму где появится сообщение о результате ( в нашем случае говорится что “ нас пока ещё нет в вашем регионе”)

1.4. Ввод данных: без согласия с условиями
ШВ:
ввести все данные корректно
убрать галку с условий
тапнуть на “Оставить заявку”
ОР:
под графой условий появится красная надпись “Для продолжения нужно согласие с условиями”
ФР:
под графой условий появится красная надпись “Для продолжения нужно согласие с условиями”

1.5. Ввод данных: дополнительное поле Страна
ШВ:
выбрать в поле “гражданство” вариант “ не имею гражданства РФ”
ОР:
появится дополнительное поле с выбором страны происхождения
ФР:
появится дополнительное поле с выбором страны происхождения

1.6. Ввод данных: Ваш регион
ШВ:
ввести в поле “ваш регион” город, пример (Нижний Новгород)
выбрать из списка, нажать ввод или перейти на друге поле
ОР:
высветиться название города
ФР:
автоматически переводится на область(регион) (в списке предложены варианты городов в регионе(Саров, Арзамас, Кстово и т. д.)) 

1.7. Ввод данных: появление кнопки "Далее"
ШВ:
выбрать регион с доставкой (пример Москва)
ОР:
кнопка "Оставить заявку" измениться на "Далее"
ФР:
кнопка "Оставить заявку" измениться на "Далее"

1.8. Ввод данных: переход на форму доставки
ШВ:
выбрать регион с доставкой
заполнить обязательные поля
нажать кнопку "Далее"
ОР: 
произойдёт переход на форму заявки на доставку
ФР:
произойдёт переход на форму заявки на доставку

1.9. Ввод данных: на Англ. языке
ШВ:
ввести в поле ФИО имя “Jack Richards” на англ. раскладке
ОР:
появится имя “Jack Richards” на английском
ФР:
появится соответствующий набор букв на русском

1.10. Ввод данных: проверка только обязательных полей
ШВ:
ввести корректные данные во все обязательные поля(обозначаются в конце “*”)
тапнуть кнопку “Оставить заявку”
ОР:
Переходит на форму где появится сообщение о результате (“ нас пока ещё нет в вашем регионе”)
ФР:
Переходит на форму где появится сообщение о результате (“ нас пока еще нет в вашем регионе”)
1.11. Ввод данных: проверка длинны строки >133 символов
Предусловие: 
в поле ФИО обязательно должен быть хотя бы один пробел
ШВ:
ввести в поле ФИО строку содержащую как минимум один пробел длинной >133 символов
убрать фокус с поля
ОР:
Под полем появится ошибка о превышении кол-ва допустимых символов
ФР:
Под полем появится ошибка о превышении кол-ва допустимых символов

1.12. Ввод данных: проверка длинны строки =133 символам
Предусловие: 
в поле ФИО обязательно должен быть хотя бы один пробел
ШВ:
ввести в поле ФИО строку содержащую как минимум один пробел длинной =133 символам
убрать фокус с поля
ОР:
появится галочка с подтвержением правилности данных
ФР:
появится галочка с подтвержением правилности данных
Тот же результат в других полях с ограниченным кол-вом символов но с другими ограничениями

1.13. Удаление из поля нажатием на крестик
ШВ: 
ввести в любое поле неправильные данные
нажать на крестик в правом углу поля
ОР:
данные в поле удаляются
ФР:
данные в поле удаляются

1.14. Копирование\вставка: Англ. раскладка
ШВ:
ввести в блокноте на ПК на английском “Jack Richards”
скопировать и вставить в поле анкеты ФИО
ОР:
появится текст на английском
ФР:
появится текст на английском

1.15. Копирование\вставка: удаление символов
ШВ:
скопировать в поле ФИО данные на англ. языке
удалить одну букву
ОР:
буква удалиться, и текст остается неизменным
ФР:
буква удаляется, а последняя буква меняется на соответствующую букву русской раскладки

2. Дополнительные проверки

2.1. Совместимость с браузерами
ШВ:
открыть анкету в Google.Браузере, Yandex.Браузере, E, Opera, FireFox
проверить тест кейсы в них
сравнить результат
ОР:
результаты тестирования будут одинаковыми во всех указанных браузерах
ФР:
результаты тестирования будут одинаковыми во всех указанных браузерах

2.2. Реакция на слабое соединение
ШВ:
смоделировать плохое соединение
заполнить заявку
отправить
ОР:
всплывает оповещение о плохом качестве соединения
ФР:
всплывает оповещение о плохом качестве соединения
Постусловие: 
при восстановлении соединения заявка успешно отправляется
